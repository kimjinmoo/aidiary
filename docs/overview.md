# AIDiary 서비스 개요 (개인정보처리방침 기초 자료)

> 본 문서는 AIDiary 앱의 개인정보처리방침 작성을 위한 기능별 수집·처리 항목 참고 자료입니다.
> 코드 변경 시 본 문서도 함께 갱신합니다.

---

## 1. 서비스 요약

| 항목 | 내용 |
|---|---|
| **서비스명** | AIDiary |
| **플랫폼** | Android 14+ (API 34), Android XR |
| **서비스 성격** | 온디바이스 AI 기반 다이어리·플래너·목표 관리 앱 |
| **핵심 원칙** | 모든 AI 처리는 기기 내부에서 수행, 사용자 데이터는 외부 서버로 전송되지 않음 |
| **개인정보처리방침** | https://www.grepiu.com/ai_diary_privacy.html |

## 2. 수집·처리하는 개인정보 항목

### 2.1 필수 수집 항목 (서비스 제공을 위해 반드시 필요)

| 수집 항목 | 수집 목적 | 보유 기간 |
|---|---|---|
| **음성 녹음 데이터** | 음성 → 텍스트 변환(STT)으로 다이어리 본문 작성 | 앱 삭제 또는 녹음 취소 시 즉시 삭제 |
| **다이어리 본문 (텍스트)** | 사용자가 작성한 다이어리 기록의 저장·조회·검색 | 앱 삭제 시까지 |
| **사진 및 영상 (첨부 미디어)** | 다이어리 기록에 이미지·동영상 첨부 | 앱 삭제 시까지 |
| **캘린더 날짜** | 다이어리 및 플래너 기록의 날짜별 관리 | 앱 삭제 시까지 |
| **위치 정보 (위도/경도/주소)** | 다이어리 기록에 현재 위치 태그 추가 (사용자가 명시적으로 요청 시에만) | 다이어리 삭제 시까지 |

### 2.2 사용자 선택 수집 항목

| 수집 항목 | 수집 목적 | 보유 기간 |
|---|---|---|
| **플래너 할 일 (텍스트, 시간, 장소, 반복 규칙)** | 사용자 일정 관리 | 앱 삭제 시까지 |
| **장기 목표 (텍스트, 카테고리, 완료 상태)** | 사용자 목표 설정 및 추적 | 앱 삭제 시까지 |
| **챗봇 대화 기록** | 온디바이스 AI 비서와의 대화 컨텍스트 유지 | 앱 삭제 시까지 |
| **AI 감정 분석 결과** | 다이어리 본문에서 자동 분류한 감정 태그 노출 | 다이어리 삭제 시까지 |
| **AI 생성 해시태그** | 본문 내용 기반 자동 생성 태그 | 다이어리 삭제 시까지 |

### 2.3 자동 생성 정보

| 항목 | 목적 |
|---|---|
| **기기 사양 정보 (RAM, GPU, SDK 버전)** | AI 모델 구동 가능 여부 판단 (서버 전송 없음) |
| **앱 사용 로그 (로컬 only)** | 없음. 앱 내에서 사용자 행동 로그, 분석, 통계 수집 일체 하지 않음 |

## 3. 권한 및 사용 목적

| 권한 | 사용 목적 | 수집 시점 |
|---|---|---|
| `RECORD_AUDIO` | 음성→텍스트 변환을 위한 마이크 녹음 | 사용자가 마이크 버튼 탭 시 |
| `CAMERA` | 카메라로 사진 촬영하여 다이어리에 첨부 | 사용자가 카메라 버튼 탭 시 |
| `ACCESS_FINE_LOCATION` / `ACCESS_COARSE_LOCATION` | 현재 위치를 다이어리 기록에 태그 | 사용자가 위치 추가 버튼 탭 시 |
| `INTERNET` / `ACCESS_NETWORK_STATE` | AI 모델 파일 다운로드 (Gemma 2.3GB, Sherpa 900MB) | 사용자가 모델 다운로드 승인 시 |
| `WAKE_LOCK` | 음성 녹음 중 화면 꺼짐 방지 | 녹음 시작 시 |

## 4. 온디바이스 AI 처리 상세

앱의 모든 AI 기능은 **Google Gemma 4 (2B 파라미터)** 모델을 기기 내부에서 직접 실행합니다.

| AI 기능 | 입력 데이터 | 출력 데이터 | 서버 전송 |
|---|---|---|---|
| **제목 자동 생성** | 다이어리 본문 텍스트 (최대 800자) | 한국어 제목 제안 | 없음 |
| **글 타입 자동 분류** | 다이어리 본문 텍스트 (최대 1500자) | 일기/새글/메모 중 분류 | 없음 |
| **본문 다듬기 (교정)** | 대상 텍스트 + 앞뒤 블록 문맥 | 교정된 텍스트 | 없음 |
| **본문 꾸미기** | 대상 텍스트 + 앞뒤 블록 문맥 | 색상·굵기·기울임·밑줄·크기 추천 JSON | 없음 |
| **한국어 번역** | 콘텐츠 텍스트 (최대 1500자) + 원본 언어 | 한국어 번역 결과 | 없음 |
| **감정 분류** | 제목 + 본문 (최대 1500자) + 날짜 | 기쁨/슬픔/분노/불안/평온 중 1개 | 없음 |
| **해시태그 추천** | 제목 + 본문 (최대 800자) | 3~5개 한국어 해시태그 | 없음 |
| **플래너 할 일 추천** | 날짜·시간·장소·반복 조건 + 기존 할 일/목표/최근 일기 | 할 일명 1건 제안 | 없음 |
| **탭별 AI 브리핑** | 최근 7건 일기(감정 포함) + 해당 날짜 할 일 + 목표 현황 | 요약 브리핑 텍스트 | 없음 |
| **AI 챗봇** | 사용자 질문 + 대화 이력 + 관련 일기/할 일/목표 | AI 응답 | 없음 |
| **목표 달성 축하** | 완료된 목표 텍스트 | 축하 메시지 | 없음 |
| **음성 인식 (STT)** | 마이크 PCM 오디오 | 한국어 텍스트 (Sherpa-Onnx 오프라인 처리) | 없음 |

## 5. 데이터 저장 및 보호

### 5.1 저장 위치

| 데이터 유형 | 저장 위치 | 접근 범위 |
|---|---|---|
| 다이어리 본문·메타데이터 | `앱 내부 저장소/diary.db` (Room DB, FTS5 인덱스) | 앱 전용 (타 앱 접근 불가) |
| 첨부 사진 | `앱 내부 저장소/diary_images/` | 앱 전용 |
| 첨부 영상 | `앱 내부 저장소/diary_videos/` (최대 30초) | 앱 전용 |
| 플래너·목표 데이터 | `앱 내부 저장소/goals.json`, `planner_tasks.json` | 앱 전용 |
| 녹음 임시 파일 | `앱 캐시 디렉토리/recording.pcm`, `recording.wav` | 앱 전용, 녹음 종료 시 삭제 |
| AI 추론 모델 | `앱 내부 저장소/models/` (Gemma), `sherpa/` (음성인식) | 앱 전용 |
| 카메라 촬영 임시 | `앱 캐시 디렉토리/capture_<ts>.jpg` | FileProvider 경유, 첨부 완료 후 내부 저장소 이동 |

### 5.2 저장 방식

- **MODE_PRIVATE**: 모든 데이터는 Android 앱 샌드박스에 저장, 타 앱 접근 불가
- **앱 삭제 시 자동 파기**: 앱 제거 시 내부 저장소의 모든 데이터가 OS에 의해 영구 삭제됨
- **DB/파일시스템에는 상대 경로만 저장**: 디바이스 간 백업·복원 호환을 위한 설계

| 구분 | 전송 여부 |
|---|---|
| **AI 처리 데이터 서버 전송** | **없음** (100% 온디바이스) |
| **사용자 콘텐츠 서버 업로드** | **없음** |
| **분석·통계·광고 데이터 전송** | **없음** |
| **사용 행동 로그 수집** | **없음** |
| **개인정보 제3자 판매** | **없음** |
| **클라우드 동기화** | **없음** (Android Auto Backup 비활성화) |

### 네트워크 통신 (유일한 예외)

사용자 승인 시에만 발생하며, 데이터를 외부로 보내지 않고 기기로 가져오기 위한 다운로드:

| 통신 목적 | 대상 서버 | 전송 데이터 |
|---|---|---|
| Gemma AI 모델 다운로드 (~2.3GB) | Cloudflare R2 (`conf.grepiu.com`) Presigned URL | HTTP User-Agent: `AIDiary/1.0` |
| Sherpa 음성인식 모델 다운로드 (~900MB) | GitHub Releases (`github.com`) | HTTP User-Agent: `AIDiary/1.0` |

- Wi-Fi 연결 확인 후 다운로드 (셀룰러 시 경고)
- 사용자가 다운로드를 취소하거나 거부할 수 있음

## 7. 사용자 권리 및 데이터 관리

### 7.1 사용자 통제 기능

| 기능 | 경로 |
|---|---|
| **개별 다이어리 삭제** | 목록 화면 → 길게 누르기 또는 상세 화면 → 삭제 버튼 |
| **첨부 사진/영상 삭제** | 다이어리 편집 화면 → 미디어 블록 삭제 버튼 |
| **플래너 할 일 삭제** | 플래너 탭 → 개별 할 일 삭제 |
| **장기 목표 삭제** | 목표 탭 → 개별 목표 삭제 |
| **챗봇 대화 내역 초기화** | 챗봇 화면 → 대화 초기화 버튼 |
| **AI 모델 삭제** | 앱 데이터 삭제 (기기 설정 → 앱 → 저장공간 → 데이터 삭제) |

### 7.2 데이터 백업 및 복원

| 기능 | 방식 |
|---|---|
| **수동 백업 (내보내기)** | 다이어리 + 플래너 + 목표 + 첨부 미디어를 ZIP 파일로 로컬 저장 |
| **수동 복원 (가져오기)** | ZIP 파일에서 데이터 복원 |

- ZIP 파일은 사용자가 선택한 위치(기기 저장소, Google Drive 등)에 저장
- 데이터는 **암호화되지 않은 상태**로 저장되므로, 사용자는 ZIP 파일 관리에 유의

### 7.3 동의 철회 방법

- 기기 설정 → 앱 → AIDiary → 권한 에서 카메라·마이크·위치 권한 개별 철회 가능
- 앱 삭제 시 모든 데이터 영구 파기

## 부록: 기능별 데이터 흐름 요약표

| 기능 | 입력(수집) 데이터 | 처리 방식 | 출력 데이터 | 저장 위치 |
|---|---|---|---|---|
| 다이어리 작성 | 제목, 본문 텍스트, 블록 콘텐츠 | Room DB 저장 | - | `diary.db` |
| 사진 첨부 | 갤러리/카메라 사진 (JPEG, HEIC, MPO) | 3D 포맷 자동 감지 → sandbox 복사 | SpatialMediaBlock (사진 경로) | `diary_images/` |
| 영상 첨부 | 갤러리 영상 (MP4, MOV, MV-HEVC, 최대 30초) | 3D 포맷 자동 감지 → sandbox 복사 | SpatialMediaBlock (영상 경로) | `diary_videos/` |
| 음성→텍스트 | 마이크 PCM 오디오 (16kHz mono) | Sherpa-Onnx 오프라인 STT → 텍스트 | 전사 텍스트 (TextBlock 추가) | 임시 캐시 (변환 후 삭제) |
| 위치 태그 | GPS 좌표 + 주소 (Geocoder) | LocationBlock 생성 | 위치 블록 | `diary.db` |
| AI 감정 분석 | 다이어리 텍스트 | Gemma 4 온디바이스 추론 | TagAiBlock (감정 태그) | `diary.db` |
| AI 제목 추천 | 다이어리 본문 | Gemma 4 온디바이스 추론 | 제목 텍스트 제안 | 작성 화면 임시 표시 |
| AI 교정·꾸미기 | 블록 텍스트 + 문맥 | Gemma 4 온디바이스 추론 | 수정 텍스트 또는 서식 JSON | 작성 화면에 적용 |
| AI 번역 | 원본 텍스트 | Gemma 4 온디바이스 추론 | 한국어 번역문 | 다이얼로그 임시 표시 |
| AI 챗봇 | 대화 이력 + 질문 + 관련 기록 | Gemma 4 온디바이스 추론 (RAG) | AI 응답 메시지 | `diary.db` (메모리·DB) |
| 플래너 관리 | 할 일 텍스트·시간·장소·반복 규칙 | PlannerRepository → JSON | - | `planner_tasks.json` |

| AI 브리핑 | 최근 기록 요약 + 할 일 + 목표 | Gemma 4 온디바이스 추론 | 요약 브리핑 텍스트 | 메모리 (1회성) |
| AI 할 일 추천 | 날짜·시간·장소·기존 할 일/목표 | Gemma 4 온디바이스 추론 | 할 일명 제안 | 메모리 (1회성) |
